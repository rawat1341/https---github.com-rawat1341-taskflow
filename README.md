#  Task Flow

**Task Flow** is a modern, interactive task management web application designed to help users organize their work efficiently. It provides authentication, task tracking, filtering, and a clean UI with light/dark mode support.

---

##  Features

*  **Authentication System** (Login / Signup)
*  **Hero Page** – Clean landing interface
*  **Dashboard**

  * Total Tasks
  * Completed Tasks
  * In Progress Tasks
  * Pending Tasks
*  **Task Management**

  * Create tasks with:

    * Title
    * Description
    * Status (Completed / In Progress / Pending)
    * Type (custom categories)
*  **Filtering System**

  * Filter tasks by **Status**
  * Filter tasks by **Type**
*  **User Task List**

  * View all created tasks dynamically
*  **Light / Dark Mode Toggle**
*  Fully interactive UI

---

##  Tech Stack

* **Frontend:** React / Next.js (assumed based on setup)
* **Backend:** Node.js + Express
* **Database:** MongoDB (assumed)

---

##  Project Setup

###  Frontend Setup

```bash
cd frontend
npm install
npm run dev
cd ..
```

---

###  Backend Setup

```bash
cd backend
npm install
npm run dev
cd ..
```

##  Usage

1. Open the application in your browser
2. Register or log in
3. Create tasks with title, description, status, and type
4. Track progress via dashboard
5. Use filters to manage tasks efficiently
6. Switch between light and dark modes
